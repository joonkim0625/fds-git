## 사용자 입장에서 기능 정리하기
- 홈페이지에 접속 후,
1. GPS
   1) GPS 정보를 기준으로 가까운 음식점 전체 리스트를 보여준다
   2) 오른쪽 상단에 필터- default: 기본 정렬순 / 별점순, 리뷰 많은순, 최고 주문 금액순, 거리순, 배달 시간순
   -> 매장 상세 페이지
       - defalut: 메뉴 / 클린리뷰, 정보
       - 메뉴 클릭 -> 가격, 선택 옵션(* 수량: default: 1. -버튼은 수량2일 때부터 작동)을 띄워줌.

       - '주문표에 추가'버튼 클릭
       - 주문표에 다른 식당 메뉴가 있는지 여부를 검사
           -> 다른 식당 메뉴가 이미 ) -> confirm창을 띄움(다른 음식점에서 이미 담은 메뉴가 있습니다. 담긴 메뉴를 취소하고 새로운 음식점에서 메뉴를 담을까요?)
           - 확인 버튼 클릭 -> '주문표에 담았습니다.'문구가 잠깐 떴다가 사라짐 -> 매장 상세 페이지 보여줌(* 새로 담은 메뉴와 정보가 주문표에 담김)
           - 취소 버튼 클릭 -> confirm창만 사라짐. 가격, 선택 옵션창으로 돌아감


       -> 매장 상세 페이지 - 화면 오른쪽에 있는 '주문표'에 '주문 내역'이 채워짐

       - '주문하기'버튼 클릭 -> 결제하기 페이지

2. 직접 주소 검색
   1) select창에 없는 주소를 입력할 경우 - '검색하신 주소를 찾을 수 없습니다.'
   2) 키보드를 입력할 때마다 매치되는 주소가 있는지 체크함

   - 입력한 주소지를 기준으로 가까운 음식점 전체 리스트를 보여준다.
   - 카테고리바 좌측에 '음식점 or 메뉴'를 입력할 수 있는 검색바를 보여준다.

   - 검색바에 음식점이나 메뉴를 검색 -> 위치 기준으로 2개씩 뜸


4. 로그인 & 회원가입
   - '로그인 | 회원가입' 버튼 클릭
       - '이메일 주소 입력(필수)' - 키보드 입력할 때마다 이메일 형식인지 체크
       - '비밀번호 입력(필수)'
   - 아이디 찾기, 비밀번호 찾기
       - '로그인'버튼
       - '네이버 아이디로 로그인'버튼

   -> 로그인 폼 & 하단에 '요기요가 처음이신가요? 이메일 회원가입'  클릭

   - 비밀번호는 8자 이상입니다.
   - 비밀번호는 숫자, 영문, 특수문자를 포함하여야 합니다.

6. 주문표에 (0)이 떠있는 경우, 버튼을 눌러서 이벤트가 발생하지 않음
   - 주문표(1)이상일 경우,  클릭 -> '주문표에 추가'버튼을 눌렀던 매장 상세 페이지를 보여줌

7. 카테고리를 클릭할 경우, 카테고리별로 리스트를 보여준다.
